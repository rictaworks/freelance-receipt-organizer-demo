# 勘定科目マスタ API

分類・手動科目変更で使用する勘定科目マスタを取得する。関連機能: **F3 勘定科目分類**。

## エンドポイント

| 項目 | 値 |
|---|---|
| メソッド | GET |
| パス | `/categories` |

## 概要

`ACCOUNT_CATEGORIES` テーブルの全 **12 件** を返す。マスタはセッションに依存しない共有データであり、セッションフィルタは適用しない（ただし取得には有効なセッションCookieを要する）。

デモ版の勘定科目（12件）: 消耗品費／旅費交通費／通信費／接待交際費／会議費／新聞図書費／水道光熱費／地代家賃／外注工賃／修繕費／租税公課／雑費。

## リクエスト

- ボディ・クエリなし。

## レスポンス（200 OK）

```json
{
  "categories": [
    { "id": 1, "code": "SHOMOHIN", "name": "消耗品費" },
    { "id": 2, "code": "RYOHI", "name": "旅費交通費" },
    { "id": 3, "code": "TSUSHIN", "name": "通信費" },
    { "id": 4, "code": "SETTAI", "name": "接待交際費" },
    { "id": 5, "code": "KAIGI", "name": "会議費" },
    { "id": 6, "code": "TOSHO", "name": "新聞図書費" },
    { "id": 7, "code": "KONETSU", "name": "水道光熱費" },
    { "id": 8, "code": "YACHIN", "name": "地代家賃" },
    { "id": 9, "code": "GAICHU", "name": "外注工賃" },
    { "id": 10, "code": "SHUZEN", "name": "修繕費" },
    { "id": 11, "code": "SOZEI", "name": "租税公課" },
    { "id": 12, "code": "ZAPPI", "name": "雑費" }
  ],
  "count": 12
}
```

`code` は科目コード（例示。実装のシード値に準拠する）。`count` は常に 12。

## エラー

フォールバックせず、マスタ未投入等の異常は明示的に返す。

| ステータス | code | 条件 |
|---|---|---|
| 500 | `MASTER_NOT_SEEDED` | 勘定科目マスタが 12 件に満たない（シード未投入・破損）。件数不整合を握りつぶさず検知する |

```json
{
  "error": {
    "code": "MASTER_NOT_SEEDED",
    "message": "勘定科目マスタが正しく初期化されていません。",
    "trace_id": "b1f2c3d4-e5a6-4789-9abc-0123456789ab"
  }
}
```

## 備考

- 分類キーワードルールマスタ（36件＝各科目3キーワード）は本 API では返さず、サーバ内部の分類処理（F3）でのみ参照する。
